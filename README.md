📌 Project Overview
The Cyber Security Awareness Portal is an educational web application built to spread awareness about common cyber threats and online safety practices.
This platform helps users understand:
Phishing attacks
Password security
Safe browsing habits
Social engineering risks
Data protection basics
🚀 Features
🏠 Home page introduction
📚 Awareness tips section
🎥 Educational videos
🧠 Interactive quiz module
📊 Multiple security levels (Level 1 – Level 5)
📝 Feedback page
🛠️ Technologies Used
HTML
CSS
JavaScript
(Optional: Node.js if server used)
🎯 Objective
To promote cybersecurity awareness and help users identify and prevent common online threats.
